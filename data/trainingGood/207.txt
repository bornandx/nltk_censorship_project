The two simplest are probably the Finite Impulse Response filter ( aka . FIR filter ) and the Infinite Impulse Response filter ( aka . IIR filter ) .\n\nThe FIR filter works by keeping a series of samples and multiplying each of those samples by a fixed coefficient ( which is based on the position in the series ) . The results of each of these multiplications is accumulated and is the output for that sample . This is referred to as a Multiply-Accumulate - and in dedicated DSP hardware there is a specific MAC instruction for doing just this.\n\nWhen the next sample is taken it\ 's added to the start of the series , and the oldest sample in the series is removed , and the process repeated.\n\nThe behavior of the filter is fixed by the selection of the filter coefficients.\n\nOne of the simplest filters that is often provided by image processing software is the averaging filter . This can be implemented by an FIR filter by setting all of the filter coefficients to the same value.\n \r\n \r\n \r\n \r\nshare|improve this answer \r\n \r\n\r\n\r\n \r\n \r\n\r\n \r\n \r\n answered Oct 5 \'08 at 2:29\r\n \r\n \r\n \r\n \r\n \r\n Andrew Edgecombe\r\n \r\n 22.5k32455\r\n \r\n \r\n\r\n \r\n \r\n \r\n\r\n \r\n \r\n\r\n \r\n \r\n\t \r\n\t\t \r\n \r\n\r\n \r\n \r\n\t\t \r\n\t \r\n\r\n \r\n\r\n add a comment | \r\n \r\n \r\n \r\n \r\n\r\n\t\t\t\t\t\t\t\t\t\r\n\t\t\t\t\t\t\r\n\t\t\t\t\t\t \r\n \r\n \r\n\t\t\t\t\t\t\tYour Answer\r\n\r\n\r\n \r\n StackExchange.ifUsing ( `` editor '' , function ( ) { \r\n StackExchange.using ( `` externalEditor '' , function ( ) { \r\n StackExchange.using ( `` snippets '' , function ( ) { \r\n StackExchange.snippets.init ( ) ;
